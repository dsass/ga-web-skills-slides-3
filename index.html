ADDENDUM
Psuedo Classes / Pseudo Selectors
  • ~Ways of adding content outside of HTML/DOM
  • ~Selectors using ":" or "::" 
  •                   
                    /* unvisited link */
                    a:link {
                      color: #FF0000;
                    }

                    /* visited link */
                    a:visited {
                      color: #00FF00;
                    }

                    /* mouse over link */
                    a:hover {
                      color: #FF00FF;
                    }

                    /* selected link */
                    a:active {
                      color: #0000FF;
                    }
                  
                
                      
                    div::before {
                      content: "before";
                    }
                    div::after {
                      content: "after";
                    }
ADDENDUM
Writing comments in code 
  •                   
                    <!-- This is a comment in html -->

                      This is a multiline comment in javascript
                      Allows you to break lines
                    */

                      This is a comment in css, used in single & multiline comments
                    */

                    p {
                      /* color: #000000; */
                      font-size: 16px;
                    }
ADDENDUM
GitHub Pages: Creating a Personal Page
  • ~To create myUsername.github.io, create a remote repo with that exact name
  • ~You must have pushed an index.html to your remote
  • ~Go to your remote on github.com and find the settings page. Scroll down for Github Pages.
  • ~Choose a branch to build the site from (most likely master branch)
